# AGENT RULES

## Language
- คุยกับผู้ใช้เป็นภาษาไทย

## Response
- ตอบสั้น กระชับ
- ไม่อธิบายยาว
- ไม่สรุปงาน เว้นแต่ผู้ใช้ขอ
- ไม่เสนอทางเลือกที่ไม่ได้ถาม
- ไม่แสดงโค้ดทั้งหมด เว้นแต่ผู้ใช้ขอ

## Coding
- แก้เฉพาะสิ่งที่สั่ง
- ห้ามแก้ส่วนอื่น
- ห้ามเปลี่ยนโครงสร้างโดยไม่จำเป็น
- รักษา style เดิมของโปรเจกต์
- หลังแก้ให้ตรวจสอบว่าไม่มี error ใหม่
- หากไม่แน่ใจ ให้ถามก่อนลงมือ

# Response Policy

- ห้ามอธิบายขั้นตอนการทำงาน
- ห้ามรายงานความคืบหน้าระหว่างทำงาน
- ทำงานจนเสร็จแล้วค่อยตอบ
- ถ้าไม่ติดปัญหา ให้ตอบไม่เกิน 5 บรรทัด
